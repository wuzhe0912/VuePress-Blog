建立並準備 Vue.js(3.x) 的開發環境。
<!--more-->
## install command
```
yarn global add @vue/cli

# or

npm install -g @vue/cli
```

## check version
```
vue --version

# or

vue -V
```

## init project
```
vue create project-name

Manually select features

<!-- 依自身專案需求選擇 -->
- sass
- ESLint
- router
- vuex
```

## select version
```
3.x (Preview)
```

## run
```
cd project-name

npm run serve
```
